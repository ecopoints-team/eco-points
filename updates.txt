# Update Log (Jan 23-24) - Sidebar & UI Fixes

### 1. Fixed the Sidebar (Finally)
- **The Flicker:** The sidebar kept closing/resetting every time I clicked a link. Turns out the `AdminLayout` was re-rendering on every single page. Refactored it to wrap everything at the root level instead. It's super smooth now.
- **Persistence:** Used `localStorage` so the sidebar actually remembers if it's open or closed when you refresh.
- **Better UX:** Added accordion-style dropdowns (so only one opens at a time) and added those cool flyout menus on hover when it's collapsed (kinda like Snipe-IT).

### 2. UI Enhancements
- **Theming:** Added a "Neutral" theme option because why not.
- **Dynamic Header:** The header title actually updates now depending on which page you're on (e.g. "Values & Permissions", "User Management").
- **Login Fix:** Fixed the icons overlapping the placeholder text on mobile.

### 3. Other Stuff
- Fixed that weird hydration error in the console.
- Cleaned up the code and removed a bunch of redundant wrappers.
- Improved the permissions UI a bit.
